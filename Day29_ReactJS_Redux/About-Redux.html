- Thư viện Redux: https://redux.js.org/ - Redux Toolkit: https://redux-toolkit.js.org/ redux là một thư viện quản lý trạng thái (state) action là một object chứa type và payload action : { type : loại action (bắt buộc) payload : dữ liệu để thay đổi state (không bắt buộc) } Một function tạo ra action thì được gọi là action creator VD: const add = () => { return { type : "add", payload : 2 } } action được gửi đến store (thông qua dispatch) để xử lý logic (thay đổi giá trị state) store : là nơi lưu trữ nhiều reducer, mỗi reducer quản lý các hàm logic một đối tượng nào đó (vd: userReducer, blogReducer, categoryReducer, v.v.) reducer : - Là một function tính toán và trả về một state mới dựa vào 2 tham số: state hiện tại, action - reducer lấy type và payload từ action để thay đổi state - Khi state thay đổi thì UI được render lại - Lưu ý: trong reducer chỉ xử lý code đồng bộ (không xử lý code bất đồng bộ, không gọi API)
